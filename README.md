# PeacePad App

This Flutter app helps users reduce overstimulation by creating awareness and gibing insights into their recovery process. The app is currently still in development and is not fully functional yet.

<img src="https://github.com/brentvanmalsen/industry-project/assets/117721826/1c2f3218-67e8-4126-bab2-396c302435f6" alt="image" width="600"/>

## Features

- **Growth Flower:** A visualisation of personal growth with a flower that grows or shrinks based on progress.
- **Overstimulation Rating:** A slider button to indicate the level of overstimulation.
- **Invite Button:** A button to invite others to use the app.
- **Progress Graph:** View a graph of your recovery progress.
- **Area Overview:** See your progress in different areas of recovery.

## Screens

## Installation

The app's pages are currently in separate branches and not yet fully functional. To test individual pages, follow these steps:

1. **Clone the repository:**
    ```sh
    git clone https://github.com/brentvanmalsen/industry-project.git
    cd overstimulation-recovery-app
    ```

2. **Checkout the desired branch:**
    ```sh
    git checkout 'branch'
    ```

3. **Install dependencies:**
    ```sh
    flutter pub get
    ```

4. **Run the app:**
    ```sh
    flutter run
    ```

Replace `branch` with the name of the branch you want to test. The branches contain different functionalities for the app.

## Getting Started

1. **Home Screen:**
   - Go to the Home Screen to get an overview of your progress and see your personal growth flower.
   - Use the big button to navigate to the rating page.

2. **Rating Screen:**
   - Use the slider to rate your current level of overstimulation.

3. **Graph Screen:**
   - View your recovery progress with the graph.

4. **Overview Screen:**
   - Check your recovery progress across different subjecst.
